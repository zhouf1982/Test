## 过程记录

> 基于mac os，其他操作系统调整相应安装方式

- 安装yarn

```bash
$ brew install yarn
```

- 项目初始化

```bash
$ cd ReactJsTest
$ yarn init
```

- 安装webpack

```bash
$ yarn add --dev webpack
```
